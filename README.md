### Принцип работы шифратора
Для примера зашифруем число 417.

##### Шаг 1. Определение стартовой позиции
Нулевая строка хранит контрольную сумму, которая служит отправной точкой шифрования.

Складываем все цифры исходного числа: 4 + 1 + 7 = 12.

Берем последнюю цифру от результата: 2.

В Строке 0 закрашиваем клетку в столбце 2.

##### Шаг 2. Каскадное шифрование цифр
Каждая последующая цифра числа шифруется в новой строке по формуле:
`Новый столбец = (Цифра + Квадрат предыдущего столбца) % 10`

**Строка 1 (Шифруем цифру 4):**
Предыдущий закрашенный столбец (в строке 0) был 2.
Квадрат: 2 * 2 = 4.
Формула: 4 (наша цифра) + 4 (квадрат) = 8.
В Строке 1 закрашиваем столбец 8.

**Строка 2 (Шифруем цифру 1):**
Предыдущий закрашенный столбец был 8.
Квадрат: 8 * 8 = 64.
Формула: 1 (наша цифра) + 64 (квадрат) = 65. Берем последнюю цифру: 5.
В Строке 2 закрашиваем столбец 5.

**Строка 3 (Шифруем цифру 7):**
Предыдущий закрашенный столбец был 5.
Квадрат: 5 * 5 = 25.
Формула: 7 (наша цифра) + 25 (квадрат) = 32. Берем последнюю цифру: 2.
В Строке 3 закрашиваем столбец 2.

### Алгоритм расшифровки для человека
Для расшифровки матрицы необходимо двигаться строго сверху вниз. Нулевая строка содержит стартовое значение, а начиная с первой строки в каждой закрашенной клетке скрыта одна цифра исходного числа.
Основная формула расшифровки:
Цифра = (Текущий столбец - Квадрат предыдущего столбца)
Если результат получился отрицательным, просто возьмите его последнюю цифру и вычтите ее из 10.
